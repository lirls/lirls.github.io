CSS3实现无缝图片滚动

2019年09月16日 427 字 大概 2 分钟 Web前端 CSS3

使用css3中animation动画实现图片无缝滚动。

效果图：

​ 其中首先所有图片的宽相加要等于盒子的宽，然后使用@keyframes animation（动画名字），在from（开始位置ps:也可以用0%）和to（结束位置ps:也可以用100%）添加transform：translateX添加动画的开始和结束X轴坐标。用animation 动画片引用动画。infinite无线循环 linear匀速。更多css元素请参考W3C官方手册。

方法：

​ 因为图片是和盒子一样宽，所以只要图片一移动后面肯定会漏出空白位置。即使动画快速回来还是会有这样的情况。所以我只需要在盒子后面在加上一组图片，这样最后一张走完了，下一排的第一张图片就会显示了；这样依次形成一个循环，超出的盒子部分的隐藏，搞定。如果理解不清楚，那么画一张图理解一下（大概是这样的QWQ）。

代码如下

HTML结构:

<nav>
    <ul>
        <li><img src="1.jpg" alt=""></li>
        <li><img src="2.jpg" alt=""></li>
        <li><img src="3.jpg" alt=""></li>
        <li><img src="4.jpg" alt=""></li>
        <li><img src="5.jpg" alt=""></li>
        <li><img src="6.jpg" alt=""></li>
        <li><img src="7.jpg" alt=""></li>
        <li><img src="1.jpg" alt=""></li>
        <li><img src="2.jpg" alt=""></li>
        <li><img src="3.jpg" alt=""></li>
        <li><img src="4.jpg" alt=""></li>
        <li><img src="5.jpg" alt=""></li>
        <li><img src="6.jpg" alt=""></li>
        <li><img src="7.jpg" alt=""></li>
    </ul>
<nav>

CSS结构:

*{
      margin:0;
      padding: 0;
     }
      li{
      list-style: none;
      }
      nav{
        width: 1050px;
        height: 100px;
        border: 1px solid palevioletred;
        margin: 300px auto;
        overflow: hidden;
       }
     nav li img{
         width: 150px;
         height: 100px;
         }
      nav li{
          loat: left;
          }
      nav ul{
           width: 200%;
           animation: moveing 3s linear infinite;
             transition: all 2s;
            }
      @keyframes moveing{
            from{
             transform: translateX(0);
             }
         to{
             transform: translateX(-1050px);
            }
       }
         nav ul:hover{
       animation-play-state:paused;
}
